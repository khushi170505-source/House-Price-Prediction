# California House Price Prediction
A Machine Learning web application that predicts California house prices using Linear Regression.

# Features
- Predict house prices using 8 input features
- Flask backend
- HTML & CSS frontend
- Trained on the California Housing Dataset
- Real-time predictions

# Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

# Dataset
California Housing Dataset (Scikit-learn)

# How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
